# Python Matplotlib Style
## [FR] Utilisation
Le style peut être utilisé avec `plt.style.use("CHEMIN/REPERTOIRE/YII")`. Ce style est prévu pour générer des figures pouvant être intégrées dans un document LaTeX au format A4 portrait avec des marges de 2 cm de chaque côté (avec `width=\textwidth`).

## [EN] Usage
The style can be used with `plt.style.use("PATH/DIRECTORY/YII")`. This style has been made to generate figures that can be integrated in LaTeX documents, with A4 paper and 2 cm margins on each side (with `width=\textwidth`).
